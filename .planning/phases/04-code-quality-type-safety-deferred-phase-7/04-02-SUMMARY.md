# Phase 4 Plan 02: Build-Time LAST_MODIFIED_DATE Codegen Summary

**Automated sitemap freshness: scripts/gen-build-meta.mjs derives LAST_MODIFIED_DATE from git log at build time via npm prebuild hook, eliminating the manually-maintained date constant**

## Performance

- **Duration:** ~3 min
- **Started:** 2026-06-19T18:04:56Z
- **Completed:** 2026-06-19T18:07:46Z
- **Tasks:** 2
- **Files modified:** 3

## Accomplishments

- Created `scripts/gen-build-meta.mjs`: an ESM Node script that runs `git log -1 --format=%cs` via `execFileSync` with a fixed argument array (no shell injection surface), validates the YYYY-MM-DD output, falls back to `new Date()` with a stderr warning if git is unavailable, and writes the date into `src/utils/constants.ts` preserving `BASE_URL` verbatim
- Wired the codegen into the build lifecycle via a `prebuild` npm hook — `LAST_MODIFIED_DATE` is now regenerated automatically before every `next build`, including Vercel deploys
- `sitemap.ts` consumer is fully unchanged — it still imports `LAST_MODIFIED_DATE` as a `string` in `YYYY-MM-DD` form and passes it to `new Date()`, so no signature change was needed

## Task Commits

Each task was committed atomically:

1. **Task 1: Create build-time codegen script for LAST_MODIFIED_DATE** - `4daca3d` (feat)
2. **Task 2: Wire codegen into npm prebuild hook** - `e9fe7bd` (feat)

## Files Created/Modified

- `scripts/gen-build-meta.mjs` - ESM codegen script: git → YYYY-MM-DD date → constants.ts (created)
- `src/utils/constants.ts` - Now has a generated LAST_MODIFIED_DATE comment + current date (modified)
- `package.json` - Added `prebuild: "node scripts/gen-build-meta.mjs"` to scripts block (modified)

## Decisions Made

- Used `execFileSync` with a fixed `['log', '-1', '--format=%cs']` array (no string interpolation, no `shell: true`) per threat model T-04-03 to eliminate command-injection surface entirely
- Fallback path uses `new Date().toISOString().slice(0, 10)` and writes to `process.stderr` with context — satisfies coding standard "no empty catch blocks" and threat T-04-04 (build never breaks)
- Generated line in `constants.ts` carries `// Generated by scripts/gen-build-meta.mjs` comment so developers know not to edit it manually
- Script reads and rewrites only the `LAST_MODIFIED_DATE` line; `BASE_URL` is preserved by parsing it from the existing file, not by hardcoding the URL
