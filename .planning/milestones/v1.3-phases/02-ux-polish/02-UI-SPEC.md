## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| gap-2 | 8px | Icon gaps, inline padding |
| gap-4 | 16px | Default element spacing, button groups |
| px-4 | 16px | Compact horizontal padding |
| px-6 | 24px | Button horizontal padding |
| py-2 | 8px | Compact vertical padding |
| py-4 | 16px | Button vertical padding |
| pt-16 | 64px | Section top padding |
| mb-2/4/6/8 | 8-32px | Vertical spacing between elements |

Exceptions: none

## Component Specifications

### CV Error Banner

**Location:** Inline, directly under Hero CTA button group (after line 81 in Hero.tsx)

**Structure:**
- Container: `div` with `flex items-start gap-2 px-4 py-3 rounded-lg`
- Icon: AlertCircle from lucide-react, size 20
- Content: Error message + optional retry label
- Auto-dismiss: 5 seconds after display

**Styling:**
```
bg-red-50 dark:bg-red-950
text-red-600 dark:text-red-400
border border-red-200 dark:border-red-800
rounded-lg
```

**Spacing:**
- From button group: `mt-4` (16px)
- Internal icon-text gap: `gap-2` (8px)
- Padding: `px-4 py-2` (16px x 8px)

**Typography:**
- Message: `text-base` (16px), normal weight
- Font color: `text-red-600 dark:text-red-400`

**Accessibility:**
- `role="alert"`
- `aria-live="polite"`
- Icon has `aria-hidden="true"`

---

### 404 Page (not-found.tsx)

**Location:** `src/app/[locale]/not-found.tsx`

**Structure:**
- Container: Full-height centered flex (`min-h-screen flex items-center justify-center`)
- Icon: FileQuestion from lucide-react, size 64
- Title: h1 with `text-4xl font-bold`
- Message: p with `text-lg text-gray-600 dark:text-gray-300`
- Button: Primary blue button matching Hero CTA style

**Layout:**
```
<div className="min-h-screen flex items-center justify-center px-4">
  <div className="text-center max-w-md">
    <FileQuestion size={64} className="mx-auto mb-6 text-gray-400" />
    <h1 className="text-4xl font-semibold mb-4">{title}</h1>
    <p className="text-lg text-gray-600 dark:text-gray-300 mb-8">{message}</p>
    <Link href={`/${locale}`}>
      <button className="px-6 py-4 bg-blue-600 hover:bg-blue-700 text-white font-semibold rounded-xl transition-colors">
        {returnHome}
      </button>
    </Link>
  </div>
</div>
```

**Spacing:**
- Icon to title: `mb-6` (24px)
- Title to message: `mb-4` (16px)
- Message to button: `mb-8` (32px)

**Typography:**
- Title: `text-4xl` (36px), `font-semibold`
- Message: `text-lg` (18px), `text-gray-600 dark:text-gray-300`

**Button:**
- Style: Primary blue (matches Hero CTA)
- Padding: `px-6 py-4` (24px x 16px)
- Rounding: `rounded-xl`

### Error Page (error.tsx)

**Location:** `src/app/[locale]/error.tsx`

**Structure:** Same as 404 page, but:
- Icon: AlertTriangle from lucide-react, size 64
- Must be `'use client'` directive (error boundaries require client component)
- Include reset button for retry functionality

**Layout:**
```tsx
'use client';

<div className="min-h-screen flex items-center justify-center px-4">
  <div className="text-center max-w-md">
    <AlertTriangle size={64} className="mx-auto mb-6 text-yellow-500 dark:text-yellow-400" />
    <h1 className="text-4xl font-semibold mb-4">{title}</h1>
    <p className="text-lg text-gray-600 dark:text-gray-300 mb-8">{message}</p>
    <div className="flex gap-4 justify-center">
      <button onClick={reset} className="px-6 py-4 bg-blue-600 hover:bg-blue-700 text-white font-semibold rounded-xl transition-colors">
        {tryAgain}
      </button>
      <Link href={`/${locale}`}>
        <button className="px-6 py-4 border border-blue-600 text-blue-600 dark:text-blue-400 dark:border-blue-400 hover:bg-blue-50 dark:hover:bg-blue-950 font-semibold rounded-xl transition-colors">
          {returnHome}
        </button>
      </Link>
    </div>
  </div>
</div>
```

**Icon color:** Yellow (warning) instead of gray
- `text-yellow-500 dark:text-yellow-400`

**Buttons:** Two-button layout with `gap-4`
- Primary: "Try Again" (calls reset prop)
- Secondary: "Return Home" (outlined style)

## Accessibility Requirements

### CV Error Banner
- `role="alert"` for screen reader announcement
- `aria-live="polite"` for non-intrusive updates
- Icon has `aria-hidden="true"` (decorative)
- Auto-dismiss timing: 5 seconds (long enough to read)

### 404/Error Pages
- Semantic HTML: `<h1>` for title, `<p>` for message
- Links/buttons have descriptive text (no "Click here")
- Keyboard navigable (tab order: button → link)
- Color contrast: WCAG AA minimum (4.5:1)
- Icons are decorative (`aria-hidden="true"`)
