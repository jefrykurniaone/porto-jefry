## Key design decisions (carried into phase CONTEXT files)

- Contrast fixed at the token level (`globals.css`), not per-component; `.readable-muted` helper untouched.
- Hero keeps all three CTAs, unified to equal height/width with `white-space: nowrap`; 1 filled primary + 2 outline.
- `sgds-switch` (theme) is already registered globally via `src/app/sgds.tsx` — no new dependency.
- Content is humanized via a `find-skills`-installed skill; no fabricated metrics; EN/ID parity enforced by `translations.test.ts`.
- Project descriptions live in i18n (bilingual), not the data file.
